# Main Flow - Texas Smart Meter Scraper
# Power Automate Desktop Flow Documentation

## Flow Overview
This is the main orchestration flow that coordinates the entire automation process.

## Detailed Flow Steps:

### Step 1: Run Subflow 'ScrapeUsage'
```
Action: Run subflow
Subflow name: ScrapeUsage
Parameters: None
Error handling: On error restart flow
```

### Step 2: Run Subflow 'ProcessFile'  
```
Action: Run subflow
Subflow name: ProcessFile
Parameters: None
Error handling: Continue on error
```

### Step 3: Run Subflow 'GPT_Injection'
```
Action: Run subflow
Subflow name: GPT_Injection
Parameters: None
Error handling: Continue on error
```

## Variables:
No global variables - each subflow manages its own variables

## Error Handling:
- ScrapeUsage: Critical - restarts entire flow on failure
- ProcessFile: Logs error and continues
- GPT_Injection: Optional enhancement - continues on failure

## Trigger:
- Manual execution
- Can be scheduled via Windows Task Scheduler
